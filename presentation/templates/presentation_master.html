    Conclusiones

    • Se diseñó una aplicación web, capaz de manejar bases de datos y el flujo de información, con Django.
    • Se utilizó la estructura interna de Django para la creación, autenticación y manejo de usuarios.
    Agenda

    Conclusiones

    • Se definió una plantilla para la creación de diapositivas web con Reveal.js
    • Se creó una estructura de control remoto de diapositivas web basado en el plúgin Multiplex.
    Agenda

    Conclusiones

    • Se desarrolló una aplicación web que unifica los dos frameworks (Django y Reveals.js) capáz de contener respuesta de usuarios.
    Agenda

    Recomendaciones
